Firefox 扩展 Auto Reload on Error

Firefox Addons：Auto Reload on Error

源代码：JPLiu/Auto-Reload-on-Error

这个 Firefox 扩展的功能很简单，当网页打开失败时，暂停几秒后刷新网页，最多尝试 3 次，已测试支持 Windows 端和 Android 端的 Firefox。这个扩展的源码，均由 AI 生成（用的腾讯元宝的 DeepSeek）。

主要用途是当 GitHub 等网站因偶发性网络问题打开失败时，自动刷新网页，话说这个功能在 Vivaldi 的 Android 端中貌似是自带的。
